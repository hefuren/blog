**1. halt 命令  关闭系统** 
```
$ halt -p  关闭系统后关闭电源；
```


**2. reboot 重新启动系统**
```
$ reboot
```

**3. shutdown  关闭并重启系统**
```
$ shutdown
```

**4. service  管理系统服务**
```
$ service smb start

$ service smb stop

$ service smb restart

```
